# drilling-ml-optimizer

Project scaffold for drilling ML optimization workflows.

## Run the Streamlit app

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Start the app from this folder:

```bash
streamlit run app.py
```

3. Open the app URL shown in terminal.

- On local machine: use `http://localhost:8501`
- In VS Code remote/container/codespace: open the forwarded port 8501 URL from the **Ports** panel

If you see "web address is invalid", make sure you are opening a full URL including `http://` and that you used `app.py` (with a dot), not `app py`.
